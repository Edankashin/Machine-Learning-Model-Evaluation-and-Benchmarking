# Machine Learning Model Evaluation & Benchmarking

## Overview
This project evaluates multiple machine learning models on structured datasets, focusing on model performance, feature importance, and predictive accuracy.

## Models Implemented
- Logistic Regression
- Support Vector Machine (SVM)
- Decision Tree
- Random Forest
- AdaBoost

## Key Features
- Train/test split with stratification
- AUC-based model comparison
- ROC curve visualization
- Feature importance using AUC drop method
- Model performance reporting (precision, recall, F1-score)

## Tech Stack
- Python
- scikit-learn
- pandas, numpy
- matplotlib

## Key Takeaways
- Compared strengths of different ML models
- Learned how ensemble methods improve performance
- Built a structured evaluation framework for model comparison
